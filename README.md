# Dairy-Management-Data-Analysis
SQL, Excel and Tableau based Dairy Management Data Analysis Project

##  Project Overview

This project analyzes dairy management data using SQL, Microsoft Excel, and Tableau to generate meaningful business insights. The analysis focuses on sales performance, revenue trends, inventory management, product shelf life, customer locations, and business profitability.


##  Objectives

- Analyze dairy sales performance
- Calculate monthly revenue
- Monitor inventory levels
- Identify top-selling products
- Track product shelf life
- Compare sales channels
- Generate business insights through interactive dashboards


##  Tools & Technologies

- Microsoft Excel
- MySQL
- Data cleaning
- Dax quaries
- Tableau
- Microsoft Word


## Dataset Information

The dataset includes:

- Product Details
